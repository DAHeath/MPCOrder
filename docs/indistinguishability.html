Indistinguishability

Definition

Definition 1 (Indistinguishability). Let \(\mathcal{L}_\ell, \mathcal{L}_r\) denote two libraries with a common interface. We say that these libraries are indistinguishable – written \(\mathcal{L}_\ell \approx\mathcal{L}_r\) – if for any polytime procedure \(\mathcal{A}\) the following quantity is negligible in \(\lambda\): \[\begin{aligned} |\mathrm{Pr}[(\mathcal{A}\diamond\mathcal{L}_\ell)(1^\lambda) \Rightarrow 1] - \mathrm{Pr}[(\mathcal{A}\diamond\mathcal{L}_r)(1^\lambda) \Rightarrow 1]| \end{aligned}\]

Properties

Indistinguishability forms an equivalence relation on libraries. More precisely, indistinguishability is trivially reflexive and symmetric, and it is also transitive.

Lemma 1 (Indistinguishability is transitive). Let \(\mathcal{L}_0, \mathcal{L}_1, \mathcal{L}_2\) be three libraries with a common interface. If \(\mathcal{L}_0 \approx\mathcal{L}_1\) and \(\mathcal{L}_1 \approx\mathcal{L}_2\), then \(\mathcal{L}_0 \approx\mathcal{L}_2\).

Proof. By triangle inequality.
